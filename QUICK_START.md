# 快速开始

欢迎使用中小学编程教育平台知识体系！本指南将帮助您快速找到所需内容。

## 🎯 5 分钟快速导航

### 我是学生 / 家长

**目标：为孩子选择合适的学习路径**

1. **第一步：确定年龄段**
   - 📖 查看 [学习路线](./docs/learning-path.md)
   - 找到对应年龄段的学习建议

2. **第二步：选择学习平台**
   - 📖 查看 [教育资源](./docs/resources.md)
   - 选择适合的在线平台或工具

3. **第三步：获取学习材料**
   - 📖 查看 [教程资源](./docs/tutorials.md)
   - 找到视频、图文教程

**推荐路径**：
- **6-8 岁**：ScratchJr → Scratch → Code.org
- **9-12 岁**：Scratch 进阶 → Python 入门 → Web 开发
- **13-15 岁**：Python 进阶 → 算法学习 → 项目实战
- **16-18 岁**：专项深入 → 竞赛 / 开发 → 作品集
