# 项目结构

```
k12-coding-education/
├── README.md                    # 项目主文档
├── LICENSE                      # MIT 许可证
├── CONTRIBUTING.md              # 贡献指南
├── PROJECT_STRUCTURE.md         # 项目结构说明（本文件）

│
└── docs/                        # 文档目录
    ├── README.md                # 文档目录说明
    ├── knowledge-system.md      # 知识体系
    ├── resources.md             # 教育资源
    ├── competitors.md           # 竞品分析
    ├── open-source.md           # 开源项目
    ├── tech-stack.md            # 技术栈

```

## 文件说明

### 根目录

#### README.md
- **内容**：项目总览、目录、快速导航
- **作用**：项目首页，GitHub 展示页面
- **包含**：
  - 项目简介
  - 目录结构
  - 知识体系概览
  - 教育资源列表
  - 竞品分析摘要
  - 开源项目推荐
  - 技术栈建议
  - 业务模块概述

#### LICENSE
- **内容**：MIT 开源许可证
- **作用**：定义项目使用权限
- **特点**：
  - 允许商业使用
  - 允许修改和分发
  - 需保留版权声明

#### CONTRIBUTING.md
- **内容**：贡献指南
- **作用**：指导社区贡献
- **包含**：
  - 如何报告问题
  - 如何提交 PR
  - 文档规范
  - 提交信息规范
  - Review 流程

#### .gitignore
- **内容**：Git 忽略规则
- **作用**：排除不需要版本控制的文件
- **包含**：
  - 系统文件
  - IDE 配置
  - 依赖目录
  - 临时文件

### docs/ 文档目录

#### README.md
- **内容**：文档目录导航
- **作用**：帮助用户快速找到所需文档
- **特点**：
  - 按角色推荐阅读顺序
  - 按需求分类导航
  - 文档概览

#### knowledge-system.md
- **内容**：完整的编程教育知识体系
- **章节**：
  - 编程语言体系（图形化 / 代码）
  - 计算机科学基础
  - 应用领域
  - 能力培养
  - 学科整合（STEAM）
- **字数**：约 8000 字
- **适合**：教育工作者、课程设计者

#### resources.md
- **内容**：全面的教育资源集合
- **章节**：
  - 在线学习平台（国际 / 国内）
  - 编程工具（在线 / 桌面）
  - 教材与书籍
  - 视频课程
  - 编程社区
  - 硬件资源
  - 竞赛资源
- **字数**：约 7000 字
- **适合**：学习者、教师、家长

#### competitors.md
- **内容**：深度竞品分析
- **章节**：
  - 国际竞品（Code.org、Scratch、Tynker 等）
  - 国内竞品（编程猫、核桃编程、小码王等）
  - 商业模式分析
  - 核心竞争力对比
  - 用户体验分析
  - 技术架构对比
  - 发展趋势
- **字数**：约 10000 字
- **适合**：产品经理、创业者、投资人

#### open-source.md
- **内容**：精选开源项目推荐
- **章节**：
  - 图形化编程平台（Scratch、Blockly）
  - 在线编程环境（Monaco、CodeMirror）

  - 代码评测系统（Judge0）
  - AI 编程助手

  - 开发工具库
- **字数**：约 6000 字
- **适合**：开发者、技术选型者

#### tech-stack.md
- **内容**：全栈技术选型指南
- **章节**：

  - 数据库选型
  - 代码执行引擎
  - DevOps 工具
  - AI / 机器学习
  - 架构设计
  - 推荐技术栈组合
- **字数**：约 8000 字
- **适合**：技术负责人、开发团队

#### business-modules.md

    - 练习评测

  - 互动教程
  - 编程挑战平台

  - 学习技巧

- **适合**：自学者、教师备课
