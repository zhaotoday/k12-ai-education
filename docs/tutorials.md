## 学习技巧

### 高效学习方法
1. **主动学习**
   - 动手实践
   - 项目驱动

2. **费曼学习法**
   - 教授他人
   - 写技术博客

3. **刻意练习**
   - 专注弱点
   - 及时反馈

4. **间隔重复**
   - Anki 记忆卡片
   - 定期复习

### 学习路径建议
1. **基础阶段**
   - 选择一门语言深入学习
   - 完成互动教程
   - 做小项目

2. **进阶阶段**
   - 学习数据结构和算法
   - 刷 LeetCode
   - 参与开源项目

3. **高级阶段**
   - 系统设计
   - 架构模式
   - 领域深耕
