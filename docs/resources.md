## 在线学习平台

### 综合性平台（国际）

#### Code.org
- **网址**：https://code.org/
- **适用年龄**：4-18 岁
- **特色**：
  - 完整的 K-12 课程体系
  - Hour of Code 编程一小时活动
  - 多语言支持（包括中文）
  - 完全免费
- **课程内容**：
  - CS Fundamentals（计算机科学基础）
  - CS Discoveries（计算机科学探索）
  - CS Principles（计算机科学原理）
  - App Lab、Game Lab、Web Lab

#### Scratch
- **网址**：https://scratch.mit.edu/
- **适用年龄**：8-16 岁
- **特色**：
  - MIT 开发的图形化编程平台
  - 全球最大的儿童编程社区
  - 6000 万+ 注册用户
  - 支持项目分享和 Remix
- **学习资源**：
  - [Scratch 官方教程](https://scratch.mit.edu/ideas)
  - [ScratchJr](https://www.scratchjr.org/)（5-7 岁）
  - [Scratch Wiki](https://en.scratch-wiki.info/)

#### Khan Academy
- **网址**：https://www.khanacademy.org/computing
- **适用年龄**：全年龄段
- **特色**：
  - 非营利教育平台
  - 完全免费
  - 自适应学习系统
  - 涵盖计算机科学全领域
- **课程分类**：
  - 计算机编程
  - 计算机科学
  - 数据结构与算法
  - 信息论

#### Codecademy
- **网址**：https://www.codecademy.com/
- **适用年龄**：13+ 岁
- **特色**：
  - 互动式编程学习
  - 即时代码反馈
  - 职业路径规划
  - 实战项目
- **课程语言**：
  - Python、JavaScript、HTML/CSS
  - Java、C++、SQL
  - React、Node.js、Django

#### Coursera
- **网址**：https://www.coursera.org/
- **适用年龄**：高中以上
- **特色**：
  - 全球顶尖大学课程
  - 专业证书和学位
  - 中文字幕支持
- **推荐课程**：
  - Python for Everybody（密歇根大学）
  - CS50（哈佛大学）
  - Machine Learning（斯坦福大学）

#### edX
- **网址**：https://www.edx.org/
- **适用年龄**：高中以上
- **特色**：
  - MIT、哈佛等名校课程
  - MicroMasters 项目
  - 免费旁听 + 付费认证
- **计算机课程**：
  - CS50's Introduction to Computer Science
  - Introduction to Python
  - Algorithms and Data Structures

#### freeCodeCamp
- **网址**：https://www.freecodecamp.org/
- **适用年龄**：初中以上
- **特色**：
  - 完全免费开源
  - 实践导向学习
  - 6000+ 小时课程
  - 社区支持
- **认证路径**：
  - Responsive Web Design
  - JavaScript Algorithms and Data Structures
  - Front End Development Libraries
  - Data Visualization
  - APIs and Microservices
  - Machine Learning with Python

## 教材与书籍

### 入门书籍

#### Python 方向
1. **《Python 编程：从入门到实践（第 3 版）》**
   - 作者：Eric Matthes
   - 适合：零基础学习者
   - 特点：项目实战导向

2. **《父与子的编程之旅：与小卡特一起学 Python》**
   - 作者：Warren Sande / Carter Sande
   - 适合：亲子学习
   - 特点：轻松有趣

3. **《趣学 Python 编程》**
   - 作者：Jason Briggs
   - 适合：8-12 岁
   - 特点：游戏和动画项目

#### JavaScript / Web 方向
1. **《JavaScript 儿童编程》**
   - 作者：Nick Morgan
   - 适合：10+ 岁
   - 特点：游戏化教学

2. **《Head First HTML 与 CSS》**
   - 适合：初学者
   - 特点：图文并茂

3. **《你不知道的 JavaScript》系列**
   - 适合：进阶学习
   - 特点：深入原理

#### Scratch 方向
1. **《动手玩转 Scratch 3.0 编程》**
   - 适合：8-12 岁
   - 特点：项目式学习

2. **《Scratch 少儿趣味编程》**
   - 适合：零基础
   - 特点：游戏制作

### 算法与数据结构

1. **《算法图解》**
   - 作者：Aditya Bhargava
   - 特点：图解清晰易懂

2. **《啊哈！算法》**
   - 作者：啊哈磊
   - 特点：趣味性强

3. **《数据结构与算法分析》**
   - 适合：高中以上
   - 特点：经典教材

### 计算机科学

1. **《计算机科学导论》**
   - 适合：高中以上
   - 特点：全面系统

2. **《编码：隐匿在计算机软硬件背后的语言》**
   - 作者：Charles Petzold
   - 特点：深入浅出

3. **《计算机程序的构造和解释（SICP）》**
   - 适合：大学水平
   - 特点：经典名著

### 创意编程

1. **《Processing 编程学习指南》**
   - 特点：艺术编程

2. **《创意编程》**
   - 特点：跨学科项目
