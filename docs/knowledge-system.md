## 编程语言体系

### 图形化编程（6-10 岁）

#### Scratch
- **适用年龄**：8-16 岁
- **核心特点**：
  - 拖拽式编程，无需输入代码
  - 丰富的角色和背景库
  - 全球最大的儿童编程社区
  - 支持项目分享和 Remix
- **学习内容**：
  - 顺序、循环、条件判断
  - 事件驱动编程
  - 变量和列表
  - 克隆和广播
  - 简单的游戏和动画制作
- **官方资源**：
  - [Scratch 官网](https://scratch.mit.edu/)
  - [ScratchJr](https://www.scratchjr.org/)（5-7 岁版本）

#### Blockly
- **适用场景**：嵌入式图形化编程
- **核心特点**：
  - Google 开发的开源库
  - 高度可定制
  - 支持代码转换（转为 Python、JavaScript 等）
- **应用案例**：
  - Code.org 的编程课程
  - App Inventor（移动应用开发）
  - Minecraft 教育版

#### Code.org
- **适用年龄**：4-18 岁全年龄段
- **核心特点**：
  - 完整的 K-12 课程体系
  - 游戏化学习关卡
  - 明星 IP 合作（Minecraft、星球大战等）
  - 支持多语言
- **课程内容**：
  - Code Studio 在线课程
  - Hour of Code 编程一小时
  - CS Fundamentals 计算机科学基础

#### 其他图形化工具
- **Tynker**：游戏化编程学习平台
- **Alice**：3D 编程环境
- **Snap!**：Scratch 的增强版本
- **mBlock**：结合硬件的图形化编程

### 代码编程（10+ 岁）

#### Python
- **适用年龄**：10+ 岁
- **为什么选择 Python**：
  - 语法简洁易读
  - 应用领域广泛
  - 丰富的学习资源
  - 强大的社区支持
- **学习路径**：
  1. **基础语法**
     - 变量和数据类型
     - 运算符和表达式
     - 控制流（if/for/while）
     - 函数和模块
  
  2. **进阶内容**
     - 面向对象编程
     - 文件操作
     - 异常处理
     - 标准库使用
  
  3. **应用方向**
     - 海龟绘图（Turtle Graphics）
     - 游戏开发（Pygame）
     - Web 开发（Flask/Django）
     - 数据分析（Pandas/NumPy）
     - 人工智能（TensorFlow/PyTorch）

#### JavaScript
- **适用年龄**：11+ 岁
- **核心优势**：
  - 前端开发必备
  - 即时可视化反馈
  - 浏览器原生支持
- **学习路径**：
  1. **JavaScript 基础**
     - 变量和数据类型
     - 函数和作用域
     - DOM 操作
     - 事件处理
  
  2. **前端三剑客**
     - HTML：网页结构
     - CSS：样式设计
     - JavaScript：交互逻辑
  
  3. **现代 Web 开发**
     - ES6+ 新特性
     - 前端框架（React/Vue）
     - Node.js 后端开发
     - 全栈开发

#### C++
- **适用年龄**：12+ 岁
- **适用场景**：
  - 信息学竞赛（NOIP/NOI）
  - 算法学习
  - 性能要求高的项目
- **学习内容**：
  - 基础语法和 STL
  - 算法和数据结构
  - OI 竞赛题目训练
  - 代码优化技巧

#### Java
- **适用年龄**：13+ 岁
- **核心特点**：
  - 面向对象编程典范
  - 企业级应用广泛
  - Android 开发
- **学习路径**：
  - Java 基础语法
  - 面向对象编程
  - 集合框架
  - 多线程编程
  - Android 应用开发

#### Swift / Kotlin
- **Swift**：iOS 应用开发
  - Swift Playgrounds 互动学习
  - SwiftUI 界面开发
  
- **Kotlin**：Android 应用开发
  - 现代化语法
  - 与 Java 兼容

## 计算机科学基础

### 1. 计算思维（Computational Thinking）

#### 问题分解（Decomposition）
- 将复杂问题拆解为小问题
- 逐步解决每个子问题
- 培养系统化思考能力

#### 模式识别（Pattern Recognition）
- 识别问题中的规律
- 发现相似性和重复性
- 应用已知解决方案

#### 抽象（Abstraction）
- 提取关键信息
- 忽略无关细节
- 建立模型和概念

#### 算法设计（Algorithm Design）
- 制定解决步骤
- 优化解决方案
- 评估算法效率

### 2. 算法与数据结构

#### 基础算法
- **排序算法**
  - 冒泡排序
  - 选择排序
  - 插入排序
  - 快速排序
  - 归并排序

- **搜索算法**
  - 线性搜索
  - 二分搜索
  - 深度优先搜索（DFS）
  - 广度优先搜索（BFS）

- **递归**
  - 递归思想
  - 递归与迭代
  - 经典递归问题

#### 数据结构
- **线性结构**
  - 数组
  - 链表
  - 栈
  - 队列

- **树形结构**
  - 二叉树
  - 二叉搜索树
  - 堆

- **图结构**
  - 图的表示
  - 图的遍历
  - 最短路径

- **哈希表**
  - 哈希函数
  - 冲突处理

### 3. 编程范式

#### 面向过程编程
- 函数和过程
- 模块化编程
- 结构化编程

#### 面向对象编程（OOP）
- **三大特性**
  - 封装
  - 继承
  - 多态
- **核心概念**
  - 类和对象
  - 属性和方法
  - 接口和抽象类

#### 函数式编程
- 纯函数
- 高阶函数
- 不可变数据

### 4. 软件工程基础

#### 版本控制
- Git 基础
- GitHub 使用
- 团队协作

#### 调试技巧
- 断点调试
- 日志输出
- 单元测试

#### 代码规范
- 命名规范
- 注释规范
- 代码风格
